## דרישות המשימה

### 1. קבצים שעליכם ליצור
צרו תיקייה חדשה בשם הפרויקט שלכם (לדוגמה: `אתר-ישיבה` או `פרסום-ישיבה`).

בתוך התיקייה צרו:
- **לפחות 2 קבצי HTML** (לדוגמה: `index.html` ו-`about.html`)
- **תיקיית `css`** עם קובץ `style.css` אחד לפחות

### 2. תגיות HTML שחובה להשתמש בהן
ודאו שהקוד שלכם כולל את התגיות הבאות (לפחות פעם אחת):

#### מבנה בסיסי:
- `<!DOCTYPE html>` - הצהרת סוג המסמך
- `<html lang="he" dir="rtl">` - תג פתיחה עם כיווניות לעברית
- `<head>` - חלק העליון (לא נראה למשתמש)
- `<body>` - התוכן שנראה בדף

#### תגי מבנה (Semantic HTML):
- `<header>` - כותרת עליונה של האתר
- `<nav>` - תפריט ניווט
- `<main>` - התוכן המרכזי של הדף
- `<section>` - קטע או חלק בתוך הדף
- `<article>` - פריט תוכן עצמאי (חדשה, מאמר)
- `<footer>` - תחתית האתר

#### תגי תוכן:
- `<h1>`, `<h2>`, `<h3>` - כותרות בגדלים שונים
- `<p>` - פסקת טקסט
- `<a href="">` - קישור לדף אחר
- `<img src="" alt="">` - תמונה
- `<ul>` ו-`<li>` או `<ol>` ו-`<li>` - רשימה

#### תגים נוספים (רצוי):
- `<figure>` ו-`<figcaption>` - תמונה עם תיאור
- `<time>` - תאריך
- `<form>`, `<input>`, `<textarea>`, `<button>` - טופס (רק אם בחרתם בדף נחיתה)

### 3. CSS שחובה להשתמש בו
הקובץ `style.css` שלכם צריך לכלול לפחות:

#### עיצוב כללי:
```css
/* צבעים לכל האתר */
body {
  font-family: Arial, sans-serif;
  background: /* צבע או גרדיאנט */;
  color: /* צבע טקסט */;
  margin: 0;
  padding: 20px;
  direction: rtl;
}
```

#### Box Model - שימוש בכל המרכיבים:
כל `.card` או תיבה צריכה לכלול:
- `padding` - רווח פנימי
- `margin` - רווח חיצוני
- `border` - גבול
- `border-radius` - פינות מעוגלות

#### עיצוב ויזואלי:
- `background` - צבע רקע (לפחות 2 שימושים שונים)
- `color` - צבע טקסט
- `text-align` - יישור טקסט
- `box-shadow` - צל לתיבות

#### פריסה (Layout):
השתמשו לפחות באחד:
- `display: flex` עם `flex-direction` ו-`gap`
- או: `display: grid` עם `grid-template-columns`

#### הובר (Hover):
```css
.main-nav a:hover {
  /* שינוי במראה כשעוברים עם העכבר */
}
```

#### רספונסיביות (אופציונלי - בונוס!):
```css
@media (min-width: 720px) {
  /* שינויים במסכים רחבים */
}
```

## טיפים וכלים שיעזרו לכם

### תמונות זמניות
אם אין לכם תמונות, השתמשו ב-Placeholder:
```
https://via.placeholder.com/800x400/003366/ffffff?text=כותרת+כאן
```
שנו את המספרים (רוחב x גובה) והצבעים לפי הצורך.

### בחירת צבעים
- השתמשו ב-[Coolors.co](https://coolors.co/) ליצירת פלטת צבעים
- או הגדירו משתנים ב-CSS:
```css
:root {
  --main-color: #003366;
  --light-color: #e6f2ff;
  --text-color: #111111;
}
```
ואז השתמשו: `color: var(--main-color);`

### מבנה המלצה לדף HTML:
```html
<!DOCTYPE html>
<html lang="he" dir="rtl">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>שם האתר שלי</title>
  <link rel="stylesheet" href="css/style.css">
</head>
<body>
  <header>
    <!-- תפריט וכותרת -->
  </header>
  
  <main>
    <!-- התוכן המרכזי -->
  </main>
  
  <footer>
    <!-- זכויות יוצרים -->
  </footer>
</body>
</html>
```

## שלבי עבודה מומלצים

### שלב 1: תכנון (5 דקות)
- בחרו אופציה (חדשות או דף נחיתה)
- רשמו על נייר: אילו עמודים? מה התוכן?
- שרטטו סקיצה פשוטה של המראה

### שלב 2: מבנה HTML (15-20 דקות)
- צרו את קובץ ה-`index.html` הראשון
- בנו את המבנה הבסיסי עם כל התגיות הנדרשות
- הוסיפו תוכן אמיתי (טקסטים, כותרות, תמונות)

### שלב 3: CSS בסיסי (15-20 דקות)
- צרו את קובץ ה-`style.css`
- הגדירו צבעים כלליים ל-`body`
- עצבו את ה-`header` וה-`footer`

### שלב 4: עיצוב מתקדם (20 דקות)
- הוסיפו `padding`, `margin`, `border-radius` לכרטיסיות
- צרו אפקט `hover` בתפריט
- השתמשו ב-`flexbox` או `grid` לסידור

### שלב 5: דפים נוספים (10-15 דקות)
- צרו עמוד שני (לדוגמה: אודות, צור קשר)
- וודאו שהקישורים בתפריט עובדים

### שלב 6: ליטוש (זמן שנותר)
- בדקו שהכל נראה טוב
- תקנו בעיות
- הוסיפו פרטים קטנים שמשפרים
