## 🎯 מה נלמד היום?

✅ מה זה JavaScript ולמה הוא משמש  
✅ כתיבת קוד ב-Console  
✅ משתנים: `let` ו-`const`  
✅ `console.log()` - הדפסה  
✅ `document.getElementById()` - מציאת אלמנט  
✅ `.textContent` - שינוי טקסט  
✅ `.value` - קריאה מ-input  
✅ פונקציות  
✅ `onclick` - אירועים

## 🚀 איך להתחיל?

### צעד 1: פתח דוגמה
לחיצה כפולה על אחד מקבצי הדוגמה - ייפתח בדפדפן

### צעד 2: שחק עם הכפתורים
נסה את כל הכפתורים וראה מה קורה

לחץ **F12** → עבור ללשונית **Console** → ראה איך הקוד רץ

### צעד 4: תרגל!
פתח את התרגילים ונסה לפתור אותם בעצמך
