## 💡 טיפים כלליים

### איך לפתוח קבצי HTML?
1. לחיצה כפולה על הקובץ → ייפתח בדפדפן
2. או: לחץ ימני → "Open with" → בחר דפדפן

### איך לראות את הקוד?
- פתח את הקובץ ב-VS Code
- או: במקלדת לחץ **Ctrl+U** בדפדפן

### איך לבדוק שגיאות?
- לחץ **F12** בדפדפן
- עבור ל-**Console**
- חפש טקסט אדום (שגיאות)
