## 🎯 מה נלמד היום?

✅ **if, else, else if** - קבלת החלטות  
✅ **אופרטורי השוואה:** ===, !==, >, <, >=, <=  
✅ **אופרטורים לוגיים:** && (AND), || (OR), ! (NOT)  
✅ **בוליאנים:** true ו-false  
✅ **alert, prompt, confirm** - אינטראקציה עם משתמש  

---

## 🔑 מושגי מפתח

### תנאי פשוט (if-else)
```javascript
if (age >= 18) {
    console.log("בוגר")
} else {
    console.log("קטין")
}
```

### תנאי מורכב (if-else if-else)
```javascript
if (grade >= 90) {
    console.log("מצוין!")
} else if (grade >= 80) {
    console.log("טוב מאוד")
} else if (grade >= 70) {
    console.log("טוב")
} else {
    console.log("ניתן לשפר")
}
```

### אופרטורי השוואה
```javascript
5 === 5    // שווה (true)
5 !== 3    // לא שווה (true)
5 > 3      // גדול מ (true)
5 < 10     // קטן מ (true)
5 >= 5     // גדול או שווה (true)
3 <= 5     // קטן או שווה (true)
```

### AND ו-OR
```javascript
// AND (&&) - שני התנאים צריכים להיות נכונים
if (age >= 18 && hasLicense) {
    console.log("יכול לנהוג")
}

// OR (||) - מספיק שתנאי אחד נכון
if (day === "שבת" || day === "חג") {
    console.log("יום חופש")
}
```

## 💡 טיפים חשובים

⚠️ **=== לעומת ==**
- השתמש תמיד ב-**===** (שוויון מדויק)
- אל תשתמש ב-**==** (יכול לגרום לבעיות)

🎯 **סדר בדיקת התנאים חשוב!**
- בדוק את התנאי היותר ספציפי קודם
- המלצה: תמיד מהגדול לקטן

💭 **בוליאנים**
- true = אמת (1)
- false = שקר (0)

## 🎨 אינטראקציה עם המשתמש

### alert - הצגת הודעה
```javascript
alert("שלום!")
```

### prompt - קבלת קלט
```javascript
let name = prompt("מה שמך?")
```

### confirm - שאלת כן/לא
```javascript
let answer = confirm("האם אתה בטוח?")
if (answer) {
    console.log("לחץ כן")
} else {
    console.log("לחץ לא")
}
```
